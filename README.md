# DevTrackr Backend

Backend API for a job application tracking system.

## 🚀 Tech Stack

- NestJS
- TypeScript
- MongoDB (Mongoose)
- JWT Authentication

## 📦 Features (in progress)

- User authentication (register/login)
- Job tracking system
- Analytics (coming soon)

## 🛠️ Setup

```bash
npm install
npm run start:dev
```
